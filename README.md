# 📊 Order Analysis Dashboard — Power BI

A Power BI dashboard built to track and analyze order operations across drivers, job types, and revenue trends over time.

---

## 🔍 Overview

This dashboard gives a clear picture of order performance across two months (March–April), covering job volume, revenue, payment status, and driver-level contributions.

Built using real operational data. Designed for quick decision-making without digging through raw reports.

---

## 📌 Key Metrics

| Metric | Value |
|---|---|
| Total Jobs | 593 |
| Total Order Value | $1.34M |
| Amount Paid | $445.88K |
| Avg Order Value | $2.26K |

---

## 📈 Dashboard Sections

**1. Daily Job Volume & Amount Trend**
Dual-axis line chart tracking daily job count and order amount across March and April. Useful for spotting peak days and revenue dips.

**2. Order Type Mix**
Donut chart breaking down orders by type: K/D, P/KD, Partial, Check, and Change Order. K/D dominates the mix.

**3. Driver Performance — Job Count**
Bar chart ranking all drivers by total jobs completed. Top performers: D15 (51 jobs) and D9 (50 jobs).

**4. Revenue by Driver (Top 10)**
Horizontal bar chart showing which drivers generated the most revenue. D9 leads at $130K, followed by D5 at $123K.
